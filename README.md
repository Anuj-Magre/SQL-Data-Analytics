## 🛒 Amazon USA Sales Analysis

An advanced **e-commerce analytics project** analyzing more than **20,000 sales records** from an Amazon-style transactional system.

### Key Analysis Areas

- Sales performance and revenue trends
- Customer lifetime value (CLTV)
- Product profitability analysis
- Shipping delays detection
- Inventory monitoring
- Seller performance evaluation
- Customer segmentation

### SQL Concepts Used

- Joins
- Window Functions
- Subqueries
- Aggregations
- Ranking functions
- CASE statements

## 🎬 Netflix Content Analysis

This project analyzes the **Netflix Movies and TV Shows dataset** to explore content distribution, ratings, actors, genres, and country-level trends.

The analysis solves **15 real-world business questions** using SQL.

### Key Analysis Areas

- Content distribution by type
- Most common ratings
- Country-wise content production
- Actor appearances in movies
- Genre distribution analysis
- Content classification using keywords

### SQL Concepts Used

- Window Functions
- JSON_TABLE
- Date Functions
- String Manipulation
- Aggregation
- Subqueries

🔗 **Project Repository**  
https://github.com/Anuj-Magre/SQL-Data-Analytics/tree/main/netflix-sql-data-analysis
